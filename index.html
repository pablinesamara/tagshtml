O Que é o HTML

Voltar

Html é uma Linguagem utilizada frequentemente na construção de sites e considerada linguagem de marcação onde cada tag indica a localização e formato do seu conteudo seja ele como textos imagens videos links e etc

Quais as principais tags

Voltar

As tag's mais utilizadas em html são:

  • Html

    • Que define todo o documento HTML aonde começa e termina todo onteudo inclusive configurações do site.

  • Head

    contém metadados sobre o título e a codificação de caracteres da página.Informações como links de bibliotecas , tipo de acentuação , titulo descrito na aba

  • Body

    • inclui todo o conteúdo que aparece na página. Contém tudo que esta dentro do site desde o cabecalho ao rodapé

  • Title

    • Define os dados exibidos na aba do navegador

  • H1,H2,H3, H4, H5, H6

    • Define um título do conteúdo sendo H1 o de maior Tamanho e prioridade seguindo até o H6 com menor sucessivamente

  • < p >

    • Essa tag define um paragrafo de conteúdo.

  • Strong

    • Utilizada para deixar os textos em Negrito

  • < U >

    • Utilizada para sublinhar um texto ou apenas uma palavra

  • < i >

    • Utilizada para deixar o testo em Italíco

  • Input

    • Utilizada frequentemente para criar caixas de textos muitas vezes para cadastro de informações

  • UL / OL

    • Utilizadas para criar listas ordenadas(OL) ou não ordenadas(UL)

  • LI

    • Onde são inseridos os itens da lista criada

  • a

    • Essa tag determina que aquele contéudo possui um link que pode ou não ser direcionado a outro site ou dentro da propria pagina
